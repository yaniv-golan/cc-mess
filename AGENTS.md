# AGENTS.md

## Project overview

**cc-mess** is an inter-Claude-Code communication plugin (design phase, not yet implemented). It aims to enable multiple Claude Code instances to form a mesh network for discovering each other, communicating, delegating tasks, and developing emergent trust relationships. See `README.md` for details.
